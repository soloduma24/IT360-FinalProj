## Project Title  
FTP Log Analysis and Threat Detection Using Splunk

---

## Overview  

### Project Description  
This project focuses on building a centralized log monitoring system using Splunk to analyze FTP server activity. The system will ingest FTP logs, extract meaningful data, and detect potential security threats such as brute force login attempts, unauthorized access, and unusual file transfers.

### Project Timeline Context  
The project proposal was submitted on **February 10, 2026**, marking the initial phase. This plan covers the execution phase from **March 31, 2026 to May 1, 2026**.

### Objectives  
- Configure FTP server to generate logs  
- Ingest FTP logs into Splunk  
- Extract and normalize log data  
- Develop detection rules  
- Create dashboards  
- Configure alerts  

### Success Criteria  
- FTP logs successfully ingested into Splunk  
- At least 3 detection use cases implemented  
- Functional dashboard created  
- Alerts triggered for suspicious activity  
- Full documentation in GitHub  

### Dates  
- Proposal Start: February 10, 2026  
- Execution Start: March 31, 2026  
- End Date: May 1, 2026
